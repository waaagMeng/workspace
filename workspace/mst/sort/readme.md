- 最简单的排序算法
 冒泡 bubbleSort
 排序的过程，两两相比，如果j小于i位置的数，就互换i,j
 两重循环
 一重循环之后，我们一定可以得到最小值在相应的位置上
 每次解决一个位置 
 i = 0  i < length - 1  i++
 j = i + 1  j < length  j++
 时间复杂度 n^2

- 冒泡排序，相邻元素的比较
 一次外层循环 将一个数排好
 每次内层循环要执行 len-i-1
 时间复杂度 n^2
